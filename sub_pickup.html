How to use

장바구니에 담은 상품을 픽업 서비스로 이동

가까운 매장 또는 원하는매장을 선택하고 주문한다.

상품 준비완료 안내 후 3일 내 매장 방문 후 QR코드를 보여준다.

PICK UP!

Check Point

  • Check Point 1.

    일부매장에 한해서 진행 중인 서비스입니다.

  • Check Point 2.

    3일 이내 매장 방문이 어렵다면, 하루 연장이 가능합니다. 매장 방문이 어려울 시 마이페이지 내 주문내력에서 자동취소 연장을 신청해주세요.

  • Check Point 3.

    20시 이전 주문할 경우, 준비 완료 메시지 확인 후 수령 가능합니다. 매장별 영업시간이 상이하므로 매장 방문 전 꼭! 확인해주세요.

  • Check Point 4.

    주문 완료 후 픽업 매장 변경은 어렵습니다. 매장 변경을 원하는 경우, 취소 후 재결제 부탁드립니다.

  • Check Point 5.

    매장 픽업 상품은 교환이나 반품은 온라인 몰 반품 절차와 동일하게 매장 반품/ 택배 중에 선택하여 진행하시면 됩니다. 반품은 수령일로부터 15일 내 온라인 몰 > 마이페이지를 통해 접수 가능합니다. *단, 교환/ 반품 사유에 따라 별도 배송비가 부과될 수 있음.
